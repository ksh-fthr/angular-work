README

Angular CLI で作成したプロジェクトを使用して、Angular の理解を深めるためにアレコレすることを目的としたものです。
このブランチでは今まで検証した内容を取り込み､ UI を自分なりに見栄えよく整形しています

次のブランチの機能を取り込んでいます
( 一部､対応するブランチのないものものあります )

ブランチ メニュー Qiita の記事
対応するブランチなし ライフサイクル-ngOnChange の確認 [Angular] ライフサイクルメソッドをみる(ngOnChanges と ngOnInit と ngOnDestroy)
feat_life_cycle_docheck ライフサイクル-ngDoCheck の確認 [Angular] ライフサイクルメソッドをみる(ngDoCheck)
feat_life_cycle_aftercontent ライフサイクル-ngAfterContent[XX] の確認 [Angular] ライフサイクルメソッドをみる(ngAfterContentInit と ngAfterContentChecked)
feat_life_cycle_afterview ライフサイクル-ngAfterView[XX] の確認 [Angular] ライフサイクルメソッドをみる(ngAfterViewInit と ngAfterViewChecked)
feat_reference_component 子コンポーネントと外部コンテンツの参照の確認 [Angular] 子コンポーネントや外部コンテンツの参照を取得する
feat_parent_child 親子関係にあるコンポーネント間でのデータ授受の確認 [Angular] 親子関係にあるコンポーネント間でデータの受け渡しを行う
feat_share_service サービスを使ったコンポーネント間におけるデータ共有の確認 [Angular] サービスを使用してデータをコンポーネント間で共有する
feat_http_client HTTP クライアント実装の確認 [Angular] HTTPクライアント(RESTクライアント)を実装する
feat_validation Validation 機能の確認 [Angular] 標準機能を利用して validation を実現する
feat_directive 属性ディレクティブ実装の確認 [Angular] 属性ディレクティブを自作する
feat_directive_event 属性ディレクティブ実装の確認( feat_directive とページを兼用 ) [Angular] 属性ディレクティブにイベント処理を登録する
feat_directive_validation 属性ディレクティブ実装の確認( feat_directive とページを兼用 ) [Angular] 属性ディレクティブにvalidation機能を設ける
feat_switch_tab タブ実装の確認 [Angular] ngComponentOutlet を使ってタブを実装する
feat_routing ルーティング実装の確認 [Angular] ルーティング による画面遷移
feat_modal モーダル実装の確認 [Angular] ngComponentOutlet を使ってモーダルダイアログを実装する
